# backspaceEngine

Web-based 3D physics engine complete with custom graphics rendering pipeline built from scratch without OpenGL. 

Hosted with GitHub and Render at [backspaceEngine](https://aadideepchand25.github.io/backspaceEngine/)

Feel free to load up demo projects such as `demo.bef`
This can be done by typing `demo` into the `Load Scene` input box*

*Please note that due to Render restrictions, API calls may take up to a few minutes the first time.
